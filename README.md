# Morf 3.4.2 full-files build

This is the full multi-file build of Morf 3.4.2.

Upload the whole folder to GitHub Pages, keeping these files together:

- `index.html`
- `styles.css`
- `tab-switcher.js`
- `button-rescue.js`
- `morf-core.js`
- `app.js`
- `version-fix.js`

Open `index.html` as the site entry point. Do not upload only the HTML from this ZIP unless the JavaScript and CSS files stay beside it in the same folder.

This build keeps Morf 3.4.2 behavior, including the name variation fixes and the button rescue layer.
